# 企业战略体系交互式导图

这是一个用于课程作业展示的小型前端项目，内容围绕：

- 总体战略
- 业务单元战略
- 职能战略

项目包含两种视图：

- 树状思维导图：点击节点展开或收起下级条目
- 关系网络图：点击节点高亮与其有关的其他条目，适合展示战略之间的联系

## 文件说明

- `index.html`：页面入口
- `styles.css`：样式文件
- `app.js`：交互逻辑
- `data/strategy-data.json`：结构化数据源
- `data/strategy-data.js`：为了浏览器直接打开时也能显示，提供的本地数据脚本

## 打开方式

### 方式一：直接打开

直接双击 `index.html`，即可在浏览器中查看页面。

### 方式二：本地服务器打开

如果你希望后续继续扩展，或者想让 `strategy-data.json` 也参与前端读取，建议在当前目录启动一个简单静态服务器，例如：

```powershell
python -m http.server 8000
```

然后访问：

[http://localhost:8000](http://localhost:8000)

## 使用说明

- 点击顶部按钮可切换“树状思维导图”和“关系网络图”
- 在搜索框中输入关键词，可快速定位节点
- 树状图支持“全部展开”和“收起层级”
- 关系图支持滚轮缩放、拖动画布平移、点击节点高亮关联关系
- 右侧信息面板会展示当前节点的概述、关键要点、下级条目和关联节点

## 适合作业展示的用法

- 如果用于课堂展示，优先用“关系网络图”说明三大战略之间的联系
- 如果用于提交作业截图，优先用“树状思维导图”展示层级结构
- 如果老师要求补充案例，可以在当前数据结构上继续增加“国际化经营战略”和“中小企业竞争战略”节点
